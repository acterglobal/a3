Module konst::parsing

source ·
Expand description

Parsing using const fn methods.

You can use the Parser type to parse from string, more information in its documentation.

If you’re looking for functions to parse some type from an entire string (instead of only part of it), then you want to look in the module for that type, eg: primitive::parse_bool.

If you do want to parse a type fron only part of a string, then you can use Parser’s parse_* methods, or the parse_with macro.
